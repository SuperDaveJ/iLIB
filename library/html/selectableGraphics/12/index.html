Style of Writing

To help ensure that warning messages are effective, they must be issued in a timely manner and should include the following infromation about the hazard:

  • Type of Threat
  • Location
  • Timeframe
  • Source of message
  • Magnitude
  • Likelihood

Style of Writing

How you write an alert/warning message is nearly as important as what you write. Some style elements to consider when writing alert and warning messages include:

  • Specificity
  • Consistency
  • Certainty
  • Clarity
  • Accuracy

Access and Functional Needs

Effective alert messages also address the needs of persons with access and functional needs. From an access and functional needs perspective, keep the following in mind when composing a message:

  • Clear and simple language
  • Avoid non-standard language and terminology to facilitate easier text-to-speech conversion and use of screen reading devices
  • Ample text and audio to explain images/maps
