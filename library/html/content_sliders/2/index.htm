Amination Slides

There are two areas: the Statistics Summary and the Power User View (by Task).

Selecting the Power User View will let you choose a security control by the drop down.

To update the status of the selected control, first upload supporting documentation (unless it has already been uploaded). Start by selecting the Upload/Identify link.

If a document supporting the findings for the control in question has already been uploaded to SMART, select it from the window below, then select Submit. If it had not been uploaded, select the Upload File to SMART button, select Browse, then select a file. Select Submit.

Once the mandatory supporting document has been uploaded, the link will show.

Add a POC for this task

The status of the control can now be changed. Choose whether these changes should be applied to any/all of the other systems under your purview, using the Apply this to other systems link (see next step). If you are not applying the changes made to other systems, select Save. Repeat the previous steps for each assessment control task.

If you are applying the changes made (test plan link, POC and Status) to other systems select the Apply this to other systems link. Select the radio boxes for all applicable systems under AOR then select Apply. Finally, select Save at upper right.
